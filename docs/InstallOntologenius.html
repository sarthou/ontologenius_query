Install Ontologenius Query

Since Ontologenius_query is a ROS package, make sure that ROS is installed on your platform. Ontologenius_query is fully supported and maintained by ROS Kinetic Kame and ROS Melodic Morenia. Desktop ROS configuration is not required for use of ontologenius_query.

Before installing ontologenius_query, make sure to have ontologenius installed.

Clone the package on your catkin workspace:

cd ~/catkin_ws/src/

git clone https://github.com/sarthou/ontologenius_query.git

cd ..

catkin_make

Test Ontologenius_query installation

Just run rostest to check if all the features work well on your platform:

catkin_make run_tests_ontologenius_query_rostest
