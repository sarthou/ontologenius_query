Ontologenius Query

This package offers an abstraction to query ontologenius with queries similar to SPARQL. It will allow you to develop applications using ontologenius quickly and simply.

Be careful when using this abstraction as it may slow down your application. For intensive queries, we recommend using the ontoloenius API.


Ontologenius and ontologenius_query use ROS for maximum compatibility with all robotics projects.
